TuMeNy Payment Gateway for WooCommerce allows your store in Zambia to accept payments from MoMo, Airtel and Zamtel mobile money customers in Zambia.

== Description ==

TuMeNy Payment makes it easy for businesses in Zambia to accept payments from MoMo, Airtel and Zamtel. Integrate with your store today, it is simple and free.

With TuMeNy Payment for WooCommerce, you can accept mobile money payments from:

* MTN (MoMo)
* Airtel
* Zamtel

= Note =

This plugin is for TuMeNy Payment merchants in Zambia. This plugin rely on a third party system (Tumeny System) to work. You must first of all signup as a Tumeny customer (Merchant) to be able to use this plugin.

Visit: https://www.tumenypay.com for more information on how to become a Merchant

We recommend you read Tumeny Privacy Policy at https://www.tumenypay.com/privacy-policy

== Installation ==

*   Go to WordPress Admin > Plugins > Add New from the left-hand menu
*   In the search box type Tumeny WooCommerce Payment Gateway
*   Click on Install now when you see Tumeny WooCommerce Payment Gateway to install the plugin
*   After installation, activate the plugin.


= TuMeNy Payment Setup and Configuration =
*   Go to WooCommerce > Settings and click on the Payments tab
*   You'll see TuMeNy Payment listed along with your other payment methods. Click Finish Set Up
*   On the next screen, configure the plugin by setting below parameters.

1. Enable/Disable - Check this checkbox to Enable TuMeNy Payment on your store's checkout
2. Title - This will represent TuMeNy on your list of Payment options during checkout. Title is set to "Mobile Money Payment" by default, but you can change it to suit your needs.
3. Description - This controls the message that appears under the payment gateway on the checkout page. You can change the default value to your needs.
4. Base Url - Enter TuMeNy Payment base url which you can get from the TuMeNy Dashboard.
5. API Key - Enter TuMeNy Payment API Key which you can get from the TuMeNy Dashboard.
6. API Secret - Enter TuMeNy Payment API Secret which you can get from the TuMeNy Dashboard.
7. Click on Save Changes to update the settings.
